## :rocket: Sobre o desafio

Reproduzir layout de uma página de login.


### Nesse projeto foi utilizado

- **`NextJS`**

- **`Firebase`**

- **`Tailwind CSS`**

- **`Cookies`**

</br>

### Funções do projeto

[x] Login por e-mail e senha. Favor utilizar (exemplo@gmail.com, 0123456789)
[x] Login por conta Google
[x] Rotas privadas
[ ] Criar conta
[ ] Esqueceu senha
